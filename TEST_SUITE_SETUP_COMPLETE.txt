⚙️ CUSTOMIZATION (Edit test_config.py)
═══════════════════════════════════════════════════════════════════════════════

Add a New Page:
  PAGES_TO_TEST.append({
      "name": "Dashboard",
      "url": "/en/dashboard/",
      "checks": ["h1", "canvas"],
      "needs_auth": True,
  })

Add Another Viewport:
  VIEWPORTS["tablet"] = {"width": 768, "height": 1024}

Change Test User:
  TEST_USERNAME = "officer"  # or "admin"

See Browser While Testing:
  HEADLESS = False
